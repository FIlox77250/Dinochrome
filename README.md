## ⚙️ Installation

```bash
# 📥 Cloner le repo
git clone https://github.com/Filox77250/chrome-dino.git

# 🔨 Compiler avec CMake
mkdir build && cd build
cmake ..
cmake --build .

# 🎮 Lancer le jeu
./dino
```
